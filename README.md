## 📌 About the Series

**21 Days of Python** is a daily Instagram video series designed to teach Python through short, focused, and beginner-friendly content. Every day covers one key concept with real code examples you can follow along with.

Whether you're a complete beginner or brushing up on fundamentals, this series is built for you.

📲 **Follow along on Instagram:** [@your_handle](https://www.instagram.com/codeing_withaj)

---

## 📅 Series Roadmap

| Day | Topic | Concepts Covered |
|-----|-------|-----------------|
| 01  | Hello, Python! | Installation, first script, `print()` |
| 02  | Variables & Data Types | `int`, `float`, `str`, `bool` |
| 03  | User Input | `input()`, type casting |
| 04  | Operators | Arithmetic, comparison, logical |
| 05  | Strings Deep Dive | Indexing, slicing, string methods |
| 06  | Lists | Creating, indexing, modifying lists |
| 07  | Tuples & Sets | Immutability, set operations |
| 08  | Dictionaries | Key-value pairs, common methods |
| 09  | Conditionals | `if`, `elif`, `else` |
| 10  | For Loops | Iterating lists, `range()` |
| 11  | While Loops | `while`, `break`, `continue` |
| 12  | Functions | Defining, calling, parameters |
| 13  | Return Values & Scope | `return`, local vs global |
| 14  | Default & Keyword Args | `*args`, `**kwargs` |
| 15  | List Comprehensions | One-liner list building |
| 16  | File Handling | `open()`, read, write, append |
| 17  | Error Handling | `try`, `except`, `finally` |
| 18  | Modules & Imports | `import`, `from`, built-in modules |
| 19  | OOP — Classes & Objects | `class`, `__init__`, attributes |
| 20  | OOP — Methods & Inheritance | Instance methods, subclasses |
| 21  | Mini Project 🎉 | Putting it all together |

## 🚀 How to Use This Repo

1. **Clone the repository**
   ```bash
   git clone https://github.com/your_username/21-days-of-python.git
   cd 21-days-of-python
   ```

2. **Navigate to any day**
   ```bash
   cd day_01
   python script.py
   ```

3. **Follow the videos** on Instagram for the full explanation, then experiment with the code!

> **Tip:** Try modifying each script to test your understanding. Breaking things is learning! 😄
